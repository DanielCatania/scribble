# Install

## Prerequisites

Make sure you have git, yarn and node installed

## Steps

1. Clone the repository project

```
  git clone https://github.com/DanielCatania/scribble.git
```

2. Access the project folder

```
  cd scribble
```

3. Install dependencies

```
  yarn install
```

4. Execute the project

```
  yarn dev
```
