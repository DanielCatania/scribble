# Commitizen

For make better project organization we use the commitizen, a tool of terminal that standardize commits

## Use mode

1. In the terminal use the command:

```
  yarn commit
```

2. Select the option that best fit your commit

<img src={ImageWithOptions.src} alt="Example of Options" />

3. Write the commit scope

<img src={ImageScope.src} alt="Scope Message Example" />

4. Write the commit name

<img src={ImageName.src} alt="Write Name Message Example" />

5. Write the commit description

<img src={ImageDescription.src} alt="Write Description Message Example" />

6. Select whether there was a breaking change or an issue resolution

<img src={ImageBreakingChange.src} alt="Breaking Change Message Example" />
<img src={ImageIssue.src} alt="Issue Message Example" />
