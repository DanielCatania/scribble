## 📁 Public Folder

Store static assets, such images, fonts, etc.

## 📁 Src Folder

Hold all the application code, this separates the other codes, such as configuration and documentation, which are at the root.

### → 📁 Pages Folder

Folder that manages application routing, the Next JS uses every file as a route.

### → 📁 Screens Folder

Hold codes related to the layout of the screens, with each folder being a screen.

### → 📁 Style Folder

Hold the style codes, such global style and theme.

### → 📁 Type Folder

Hold global types of the project.

### → 📁 Stories Folder

Hold storybook files, global documentation files.


### → 📁 Patterns Folder

Hold patterns, view what are patterns in patterns section.

### → 📁 Components Folder

Hold components, view the difference between patterns and components in patterns section.


## 📁 .storybook Folder

Storybook folder, hold config storybook files.

## 🗎 .editorconfig File

Standardization file for editor configuration.

## 🗎 next.config.js File

Next JS Settings File.

## 🗎 next-env.d.ts File

File that inserts Next JS types into the Typescript compiler.
