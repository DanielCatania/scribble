# Install/Instalar

## Prerequisites/Pré-requisitos

Make sure you have git, yarn and node installed

> Certifique-se de ter git, yarn e node instalados

## Steps/Passos

1. Clone the repository project

> 1. Clone o repositório do projeto

```
  git clone https://github.com/DanielCatania/scribble.git
```

2. Access the project folder

> 2. Acesse a pasta do projeto

```
  cd scribble
```

3. Install dependencies

> 3. Instale as dependências

```
  yarn install
```

4. Execute the project

> 4. Execute o projeto

```
  yarn dev
```
